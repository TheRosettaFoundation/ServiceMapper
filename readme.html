MT-Broker will interact with 'locConnect' for 'SOLAS' will have two parts: MTBroker_Select and MTBroker_Connect.
In future MTBroker_Select sends back to locConnect a nominated service descriptor considered as an optimal service for the provided workflow requirements.
Then locConnect call MTBroker_Connect to call the target MT.
On 2011/Nov only MTBroker_Connect has been developed connecting Babelfish MT provided by Yahoo!. MTBroker_Select is under development.
